# Cut & Build — Fitness Plan

A back-safe, home-friendly 8-week fitness and nutrition app built with React + Vite.

## Deploy to Railway

1. Push this folder to a GitHub repo
2. Go to [railway.app](https://railway.app) → New Project → Deploy from GitHub
3. Select your repo — Railway will auto-detect the config and deploy
4. Your app will be live at the Railway-provided URL in ~2 minutes

## Run Locally

```bash
npm install
npm run dev
```

Then open http://localhost:5173
